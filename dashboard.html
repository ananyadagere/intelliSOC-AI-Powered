BACKEND CONNECTION GUIDE
1
Install dependencies
In your project folder, activate venv then install: pip install fastapi uvicorn lightgbm scikit-learn shap pandas numpy joblib requests faker
2
Enable CORS in api/main.py — add this after app = FastAPI(...): from fastapi.middleware.cors import CORSMiddleware app.add_middleware(CORSMiddleware, allow_origins=["*"], allow_methods=["*"], allow_headers=["*"])
3
Start the backend (from your project root): uvicorn api.main:app --reload --port 8000 You should see: Application startup complete.
4
Start Ollama (for playbook generation): ollama serve # In another terminal: ollama pull phi3:mini
5
Open this dashboard — open the HTML file directly in browser, or serve with: python -m http.server 3000 # Then open: http://localhost:3000
6
Click "TEST CONNECTION" at the top — if the dot turns green, you're connected. If red, check that your backend is running and the URL is correct (default: http://localhost:8000).
